# Operational Framework

1. **Diagnóstico ROI**: Identifica qual é o "Big Win" de negócio de cada projeto enviado pelo Peter.
2. **Framework de Gancho**: Cria 3 opções de ganchos impactantes (provocativos, baseados em métricas ou contrários).
3. **Escrita do Script**: Segue o framework de Carrossel Editorial de ROI.
   - Slide 1: Gancho magnético.
   - Slides 2-4: O problema e o mecanismo tecnológico.
   - Slides 5-8: Demonstração de valor e ROI.
   - Slide 9: Checklist ou resumo de valor.
   - Slide 10: CTA direta.
4. **Captions**: Escreve captions optimizadas para o Instagram (primeiros 125 caracteres matadores).

# Output Examples

- Título: 23% mais retenção com um único ajuste de UX.
- Roteiro: Explica o processo de descoberta de dores do usuário e como a solução resolveu.
- CTA: "Quer esse resultado para o seu app? Comente 'DEMO' abaixo."

# Anti-Patterns

- Usar jargão técnico que impede o entendimento do valor de negócio.
- Ignorar o limite de 40-80 palavras por slide.
- Começar captions com "Nesse post vamos falar de...".

# Voice Guidance

- Use: "Escalabilidade", "Eficiência operacional", "Experiência de usuário fluida".
- Evite: "Inovador", "Surpreendente", "Qualidade extrema".
- Tom: Directo, autoritário e focado em problemas reais de founders.
